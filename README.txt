SEASIDE BEACH TEMİZ SÜRÜM

1. ZIP'i çıkar.
2. Klasörü VS Code ile aç.
3. Terminal:
   npm install
   npm run dev
4. Ana sayfa: http://localhost:3000
5. Menü: http://localhost:3000/menu
6. Admin bilgi ekranı: http://localhost:3000/admin

Bu sürümde rezervasyon yoktur.
Fotoğraf, video, fiyat, konum, Instagram ve menü içerikleri dahildir.
Tailwind ve Supabase gerekmez.
